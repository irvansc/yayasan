Senior Web Developer

Tech Indo Jakarta Selatan 2 Hari yang lalu
Gaji Ditawarkan IDR 25.000.000 / bulan

Deskripsi Pekerjaan

Kami mencari Senior Web Developer yang berpengalaman dan bersemangat untuk bergabung dengan tim teknis kami yang sedang berkembang pesat. Anda akan bertanggung jawab untuk memimpin pengembangan aplikasi web skala besar, memastikan kode berkualitas tinggi, dan membimbing pengembang junior.

Posisi ini membutuhkan pemahaman mendalam tentang siklus pengembangan perangkat lunak modern, mulai dari konsep hingga deployment dan maintenance.

Tanggung Jawab

  • Merancang, mengembangkan, dan memelihara aplikasi web yang efisien, dapat digunakan kembali, dan andal.
  • Berkolaborasi dengan tim desain dan manajer produk untuk mendefinisikan fitur baru.
  • Mengidentifikasi hambatan kinerja dan bug, serta merancang solusi untuk masalah tersebut.
  • Melakukan code review dan memberikan feedback konstruktif kepada anggota tim.

Kualifikasi

  • Minimal 5 tahun pengalaman dalam pengembangan web profesional.
  • Menguasai HTML5, CSS3, JavaScript (ES6+), dan framework modern (React/Vue/Angular).
  • Pengalaman kuat dengan Backend (Node.js, Python, atau PHP) adalah nilai tambah.
  • Memahami prinsip RESTful API dan integrasi sistem.
  • Gelar Sarjana di bidang Ilmu Komputer atau setara.
